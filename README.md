# 🏥 MedAppoint – Doctor Appointment Booking Website

MedAppoint is a simple and user-friendly **Doctor Appointment Booking Web Application** built using **HTML, CSS, and JavaScript**.
This platform allows patients to easily book doctor appointments, enter their details, select doctors, and download an appointment confirmation ticket.

## 🚀 Features

* 👤 Patient Registration
* 🔐 Mobile Number Login System
* 🧑‍⚕️ Doctor Selection Page
* ⏰ Appointment Time Selection
* 💳 Payment Confirmation Page
* 📄 Appointment Confirmation Ticket
* 📱 Simple and User-Friendly UI (Elderly Friendly)

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **LocalStorage (for storing user data)**

## 📌 How It Works

1. User enters **patient details**
2. Selects **doctor and appointment time**
3. Completes **payment confirmation**
4. System generates **appointment confirmation**
5. User can **download appointment ticket**
